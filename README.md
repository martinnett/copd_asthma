## 系统功能（v1.0）

### 管理员
* 登录及退出
* 新建录入员
* 录入登记数据
* 按条件查询所有登记卡（列表及详情）
* 导出数据（Excel）

### 录入员
* 登录及退出
* 录入登记数据
* 查询自己录入的数据
